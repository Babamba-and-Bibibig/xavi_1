# Planning Agent Docs

이 폴더는 사용자와 많은 대화를 하며 기획과 요구사항을 정리하는 AI 에이전트 전용 문서 시스템이다.

## 담당 역할

- 요구사항 정리
- 기능 범위 조율
- 설계 방향 논의
- 사용자 의도 해석
- 의사결정 메모 정리

## 접근 허용 범위

- 이 폴더 내부 문서
- 공통 진입 문서인 `starter.md`
- 공통 인덱스인 `docs/agent/README.md`

## 접근 금지 범위

- `docs/agent/codegen/`
- `docs/agent/review/`
- `docs/agent/ephemeral/`
- `docs/human/`

## 관리 문서

- `brief.md`: 현재 프로젝트 또는 기능 브리프
- `dialogue-log.md`: 사용자와의 대화 정리
- `decision-log.md`: 합의된 결정과 미결 사항

## 원칙

- 불명확한 요구사항을 먼저 명확히 한다.
- 구현 세부사항을 섣불리 확정하지 않는다.
- 코드 리뷰 판단은 여기서 하지 않는다.
- 구현 진행 상황 기록은 여기서 관리하지 않는다.
- 사용자와 대화한 핵심 내용은 스스로 문서화한다.
- 합의된 결정과 미결 사항도 스스로 갱신한다.
- 더 이상 중요하지 않은 대화 찌꺼기는 정리하고 핵심만 유지한다.
