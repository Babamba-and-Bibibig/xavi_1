# Documentation System

이 프로젝트의 문서는 크게 두 축으로 나뉜다.

- `human/`: 사람이 읽고 판단하는 문서
- `agent/`: AI 에이전트용 문서 시스템 루트

다만 `agent/` 는 단일 문서 폴더가 아니다.
그 내부에는 역할별로 분리된 네 개의 독립 문서 시스템이 존재한다.

- `codegen/`: 코드 생성 담당 AI
- `review/`: 코드 수정 검수 및 리뷰 담당 AI
- `planning/`: 사용자와 기획 대화를 많이 수행하는 AI
- `ephemeral/`: 필요 시 임시로 생성되는 서브 AI

그리고 `human/` 아래에는 사람 독자를 위한 문서를 실시간으로 정리하는 전담 AI 세션용 영역이 따로 존재할 수 있다.

- `human/user-docs/`: 사용자 문서 전담 AI 가 관리하는 실시간 사용자 문서

공통 원칙은 다음과 같다.

- 사람용 문서와 에이전트용 문서는 목적이 다르다.
- 에이전트용 문서 시스템은 역할별로 서로 침범하지 않는다.
- 사용자 문서 전담 AI 는 `docs/human/user-docs/` 만 문서 작업 영역으로 사용한다.
- AI 세션은 `starter.md` 와 자신의 역할 문서만 읽는 것을 기본 규칙으로 한다.
