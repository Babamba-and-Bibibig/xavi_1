# Codegen Backlog

이 문서는 주제 주입 단계에서 `codegen` 이 참고할 수 있는 초기 구현 항목을 정리하는 읽기용 대기열이다.
일반 `planning` 세션과 `codegen` 세션은 이 문서를 직접 작성하거나 갱신하지 않는다.
이 문서의 갱신은 `orchestra` 가 취합한 입력을 바탕으로 `ai-docs` 역할이 맡는다.

## 사용 방식

- 구현 가능한 단위로 작업을 기록한다.
- 각 항목은 레이어 위치가 명확해야 한다.
- 불명확한 항목은 주제 주입 전에 먼저 정리하고, 여기서 추정하지 않는다.

## 초기 상태

- 아직 등록된 구현 항목 없음
