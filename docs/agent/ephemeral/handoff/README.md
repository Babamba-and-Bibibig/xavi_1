# Ephemeral Handoff

이 폴더는 `ender.md` 가 종료 유형을 `interrupted-handoff-close` 로 분류했을 때만 사용한다.
정상 사이클 종료에서는 이 폴더를 갱신하지 않는다.

작성 대상:

- `docs/agent/ephemeral/handoff/latest.md`

`ephemeral` 은 자기 인계에 아래를 남긴다.

- 임시 작업의 목적
- 실제로 확인한 내용
- 아직 확인하지 못한 내용
- 다음에 넘겨야 할 역할
- 자기 세션 폴더가 있다면 해당 경로
- 자기 `Context Report`
