# Review Findings

이 문서는 리뷰 에이전트가 발견한 이슈를 기록한다.

## 기록 규칙

- 문제를 먼저 적는다.
- 가능한 한 구체적 위치를 적는다.
- 심각도와 영향 이유를 함께 적는다.
- 단순 감상은 적지 않는다.
- 과거 cycle id, 임시 alias, 특정 pass count 는 초기 템플릿에 남기지 않는다.

## 초기 상태

- 아직 기록된 발견 사항 없음
