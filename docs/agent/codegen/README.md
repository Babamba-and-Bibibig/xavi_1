# Codegen Agent Docs

이 폴더는 코드 생성과 구현을 담당하는 AI 에이전트 전용 문서 시스템이다.

## 담당 역할

- 새 코드 작성
- 구조에 맞는 파일 생성
- 포트와 어댑터 구현
- 실행 가능한 코드 변경

## 접근 허용 범위

- `starter.md`
- `docs/agent/README.md`
- 이 파일
- 필요 시 `constraints.md`

## 접근 금지 범위

- `docs/agent/review/`
- `docs/agent/planning/`
- `docs/agent/ephemeral/`
- `docs/human/`

## 문서 규칙

- 이 역할은 문서를 작성하지 않는다.
- 이 역할은 문서를 갱신하지 않는다.
- 이 역할은 코드 생성에만 집중한다.
- 이 역할의 출력은 코드 변경뿐이다.

## 원칙

- 구현은 레이어 책임을 먼저 보고 시작한다.
- 코드 배치는 책임 기준으로 판단한다.
- 리뷰 판단은 여기서 하지 않는다.
- 사용자와 장기 기획 대화는 여기서 주도하지 않는다.
- 사용자용 문서도 작성하지 않는다.

## Validation Boundary

- 이 역할은 하네스의 주 사용자가 아니다.
- 검수와 하네스 기반 테스트의 소유권은 `review` 역할에 있다.
- 필요하면 구현 변경 과정에서 하네스 코드를 건드릴 수는 있지만, 하네스 사용 흐름을 주도하지 않는다.
- 테스트 전략과 검수 기록은 `review` 가 관리한다.
