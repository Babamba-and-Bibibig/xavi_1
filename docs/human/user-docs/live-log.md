# User Live Log

이 문서는 사용자 문서 전담 역할이 사용자에게 의미 있는 변경을 시간순으로 남기는 곳이다.

## 현재 상태

- 아직 특정 사용자 프로젝트 개발 로그는 없다.
- 이 저장소는 AI 협업 부트스트랩 초기값으로 보이도록 유지한다.
- 과거 로컬 실행 프로세스 번호, 임시 report URL, 특정 cycle id/alias, 특정 테스트 pass count 는 상시 로그에 남기지 않는다.

## 기록 항목

프로젝트 주제 주입 뒤에는 각 항목을 아래 형식으로 기록한다.

```text
## <날짜 또는 cycle placeholder>

- 대상:
  - <파일 또는 기능>
- 변화:
  - <사용자가 알아야 할 변화>
- 확인된 결과:
  - <검증 명령 또는 확인 방법>
- 주의:
  - <남은 제한 또는 다음 확인 사항>
```

## 작성 기준

- 실제 변경과 확인된 검증만 기록한다.
- 사용자용 문서이므로 내부 역할 지시 전문이나 AI 인계 메모를 복사하지 않는다.
- `development_trace` 와 cycle report 의 raw evidence 는 원장과 artifact 에 두고, 이 문서에는 사용자가 이해할 수 있는 요약만 남긴다.
