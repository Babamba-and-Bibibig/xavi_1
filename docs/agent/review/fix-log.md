# Review Fix Log

이 문서는 리뷰/수정 역할이 실제로 반영한 수정과 그에 대한 검증, 아직 남은 이슈를 누적 기록하는 문서다.

## 기록 규칙

- 무엇을 수정했는지 적는다.
- 왜 수정했는지 짧게 적는다.
- 어떤 하네스 검증을 했는지 적는다.
- 아직 남은 이슈가 있으면 적는다.
- 이미 끝난 오래된 항목은 필요하면 압축하거나 삭제한다.

## 권장 형식

### Active

- 아직 진행 중인 수정만 둔다.

### Recent

- 최근에 끝난 수정 중 다음 작업에 의미 있는 것만 짧게 남긴다.

### Cleared

- 완전히 해소되어 더 이상 참조 가치가 없으면 남기지 않는다.

## 초기 상태

- 아직 기록된 수정 로그 없음
